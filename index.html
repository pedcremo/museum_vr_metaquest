VR Gallery Escritorio: clic en la escena para capturar el ratón, WASD para caminar, ratón para mirar, F para activar/desactivar vuelo, Espacio/E para subir, Ctrl/Q para bajar, Esc para salir. Las teclas verticales reactivan el vuelo si estaba desactivado. Clic en los interruptores para encender/apagar luces. VR: stick izquierdo para caminar, stick derecho para girar a saltos, aprieta el gatillo de agarre (grip) para teletransportarte y usa los anillos dorados del suelo para colocarte delante de cada obra. Con el gatillo puedes pulsar interruptores, el botón "Acercarme" bajo cada cartela o apuntar directamente a un cuadro para colocarte automáticamente frente a él.
